## 📦 Installation

**Prerequisites**
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Git (optional, for cloning)

**Setup Instructions**

1. **Clone the repository**
```bash
git clone https://github.com/Uzi-gpu/full-stack-javascript-projects.git
cd full-stack-javascript-projects
```

2. **Launch the Portfolio**
Open the root `index.html` file directly in your web browser. This acts as a central hub linking to all the projects. No build tools or local servers are required!

## 📊 Project Details

### 1. 📝 Uzair's Dev Log
**File:** `first project/index.html`

**Objective:** Build a structurally sound, semantic personal blog.

**Key Features:**
- ✅ Strict semantic HTML tags (`<header>`, `<main>`, `<article>`, `<footer>`)
- ✅ Clean separation of content and structure
- ✅ Introduction to modern CSS styling and typography pairing
- ✅ Multi-page navigation (`index.html` and `about.html`)

### 2. 📰 The News Times
**File:** `second project/index.html`

**Objective:** Implement a complex, responsive digital newspaper layout using modern CSS techniques.

**Concepts Covered:**
- Advanced CSS Grid implementation
- Spanning rows and columns dynamically
- Typography scaling for readable articles
- Responsive design adaptation for mobile views

**Implementation:**
- ✅ 3-column CSS Grid layout
- ✅ Hover states and interactive article cards
- ✅ High-contrast, accessibility-friendly theme

### 3. ☕ Coffee Masters
**File:** `coffee shop project/index.html`

**Objective:** Create a premium, visually appealing e-commerce menu and checkout interface.

**Design Highlights:**
- **Glassmorphism UI:** Translucent backgrounds with blur filters
- **Responsive Layout:** Side-by-side flex layouts adapting to mobile screens
- **Micro-animations:** Smooth button interactions and item hovering
- **Modern Forms:** Floating labels for input fields

### 4. 🧮 iOS Style Calculator
**File:** `calculator project/index.html`

**Objective:** Build a fully interactive, state-driven calculator application.

**Mathematical & JS Concepts:**
- Variable state management (`runningTotal`, `buffer`, `previousOperator`)
- Type casting (`parseInt`) to handle DOM strings safely
- Event delegation for button clicks
- Mathematical logic and string manipulation

**Implementation:**
- ✅ Perfect CSS Grid layout mimicking standard calculators
- ✅ Functional arithmetic operations (+, -, ×, ÷)
- ✅ Clear (C) state resetting
- ✅ Backspace arrow (`←`) for single-digit deletion

### 5. 🎓 Class Dashboard
**File:** `JS assessment/index.html`

**Objective:** Process an array of complex data objects and dynamically render a dashboard into the DOM.

**Complete Pipeline:**
- **Data Parsing:** Iterating over an array of student objects
- **Logic Computation:** Calculating averages and mapping them to letter grades
- **DOM Injection:** Creating HTML elements dynamically (`document.createElement`) and injecting them into the document
- **UI Updating:** Populating summary cards with dynamic statistics (Top Student, Class Average, Honor Roll count)

**Real-World Skills:**
- ✅ Arrays and Object manipulation
- ✅ `forEach` and `reduce` array methods
- ✅ Dynamic HTML rendering via JavaScript

### 6. 🍕 React Pizza Shop
**File:** `Pizza project/`

**Objective:** Build a modern, full-stack React application integrating advanced routing and data-fetching.

**React & Full-Stack Concepts:**
- Component-based architecture and JSX
- State management with React Hooks (`useState`, `useEffect`, Custom Hooks)
- Advanced Routing using **TanStack Router**
- Data fetching and caching using **TanStack Query**
- Backend integration via a Node.js API with a SQLite database

**Implementation:**
- ✅ Uncontrolled forms for order submission
- ✅ Dynamic cart management and UI updates
- ✅ Robust error boundaries for fail-safe rendering

*(Note: Because this project uses a build tool (Vite) and a local database, it must be run locally using `npm run dev` rather than a static GitHub Pages link.)*

## 📚 Key Web Concepts Demonstrated

- **Semantic HTML:** Building for accessibility and Search Engine Optimization (SEO).
- **CSS Layouts:** Mastery of both Flexbox (1-dimensional layouts) and Grid (2-dimensional layouts).
- **DOM Manipulation:** Bridging the gap between static HTML and dynamic JavaScript logic.
- **Separation of Concerns:** Keeping structure (HTML), presentation (CSS), and behavior (JS) clean and modular.
